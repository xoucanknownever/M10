# M10 — 4th Anniversary Website

A romantic, interactive anniversary website built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Site:** [bhyresh-dev.github.io/M10](https://xoucanknownever.github.io/M10/)

---

## Features

- **3D Hero Scene** — Spline-powered interactive 3D background
- **Loading Screen** — Animated bouncing hearts while the scene loads
- **Floating Hearts & Sparkles** — Canvas-driven particle animation
- **Digital Love Letter** — Envelope open animation with typing effect
- **Long Distance Map** — SVG connection map between two cities
- **Why I Love You** — Random reason generator
- **Memory Gallery** — Polaroid-style photo grid with full-screen modal viewer
- **Background Music** — Manual toggle with fade in/out (no autoplay)
- **Easter Egg** — Hidden star with a secret message
- **Scroll Progress Bar** — Top-of-page reading indicator
- **Cursor Glow** — Soft light that follows the mouse
- **Click Effects** — Ripple, bloom, and burst hearts on every click

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [Spline](https://spline.design/) — 3D scene
- [Google Fonts](https://fonts.google.com/) — Playfair Display, Dancing Script, Lora
- GitHub Pages — Hosting

